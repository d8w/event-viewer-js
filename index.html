Porting Event Viewer to the web.

Accessing the demo website

Step 1. forward port 8443 (Linux/Mac)

ssh -L8443:compute-0-0.int.autonlab.org:8443 <your_auton_account>@lop1.autonlab.org

Windows users can download and use PuTTY.

Step 2. point your browser to https://localhost:8443/owf, log in with the following credential

testUser2
password

Step 3. accept the browser security warning. This article describes steps for Firefox, IE and Chrome(recommended).

Portal

We use Bitbucket to host the project. Event-viewer-js on Bitbucket is a private repository, so you will need an authorized Bitbucket account to access its content.

  • File bugs and request features

Browser compatibility

  • Firefox
  • Chrome
  • IE 10+

Layouts

You can access event-viewer-js in different layouts. Two predefined layouts are tab view and portal view. You can switch between them by clicking My Apps -> EventViewer or EventViewer Portal. You can add your own layout by clicking Create New in My Apps dialog.
